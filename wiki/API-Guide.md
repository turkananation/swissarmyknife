# API Guide

SwissArmyKnife exports:

- Extensions for strings, collections, maps, numbers, dates, futures, streams,
  URIs, regex, bytes, nullable values, and related helpers.
- Functional types: `Result`, `Option`, `Either`, `Pair`, `Triple`, and
  pipelines.
- Async helpers: retry, debounce, throttle, and rate limiters.
- Pattern helpers: validator, event bus, disposal, lazy values, commands, and
  state machines.
- Infrastructure: HTTP/API client, safe JSON, schema validation, cache, env,
  logging, date ranges, recurrence, cron, and benchmarking.
- Advanced utilities: memoization, circuit breaker, task queue, tree/graph
  helpers, codec pipeline, expression evaluator, reactive store, middleware.
