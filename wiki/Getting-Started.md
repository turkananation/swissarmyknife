# Getting Started

Add the package:

```yaml
dependencies:
  swissarmyknife: ^0.1.0
```

Then import the public barrel:

```dart
import 'package:swissarmyknife/swissarmyknife.dart';
```

Use SwissArmyKnife for recurring utility glue: expressive extensions, explicit
result types, validation, async control, safe JSON, HTTP helpers, and small
architecture primitives.
